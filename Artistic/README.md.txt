## About The Project

## Artistic - Gallery Viewing App

A application for Artists to share and view work.

### Functionality Covered

1. Register
2. Login
3. Browse Items
4. View individual Item

Additional Elements
1. Added Light and Dark Themes
2. Added Flyout Navigation
3. Added sqlite db for user auth

### Built With

* .Net Maui
* sqlite

Packages:
sqlite-net-pcl
SQLitePCLRaw.bundleGreen 
community-toolkit-mvvm
